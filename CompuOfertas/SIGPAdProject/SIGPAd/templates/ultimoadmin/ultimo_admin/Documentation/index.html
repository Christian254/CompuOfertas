HTML Structure

This template has two sections, the header, the sidebar, and the main container. 

Header Contains:
  • Logo
  • Search Box
  • Notifications, Messages and User
  • User Functions (Setting, Profile, Message, Logout)
  • Logout
Sidebar Contains:
  • Main Navigation
  • Container contains all the necessary codes
